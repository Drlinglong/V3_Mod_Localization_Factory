## 使用说明

### 支持的编辑器

**推荐使用以下编辑器来方便地使用复选框功能：**

1. **VS Code / Cursor**
   - 原生支持Markdown复选框
   - 点击 `[ ]` 自动切换为 `[x]`
   - 有丰富的Markdown预览插件

2. **Typora**
   - 专门为Markdown设计的编辑器
   - 复选框交互体验最佳
   - 所见即所得的编辑模式

3. **Obsidian**
   - 强大的知识管理工具
   - 完美支持Markdown复选框
   - 支持任务管理功能

4. **GitHub Web界面**
   - 直接在GitHub上编辑
   - 复选框支持良好
   - 支持版本控制

5. **Notion**
   - 在线协作工具
   - 支持任务列表和复选框
   - 可以导入Markdown

### 多语言支持

- 系统会根据用户选择的目标语言自动生成对应版本的Markdown文件
- 支持的语言：中文、英文、法文、德文、西班牙文
- 如果选择不支持的语言，会自动回退到英文版本
- 所有文本内容（标题、列名、说明等）都会自动翻译
